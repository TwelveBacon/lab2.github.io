У ході виконання лабораторної роботи №2 було розроблено три екземпляри головної сторінки веб-сайту на основі варіанту №16 із попередньої роботи. 
Кожна сторінка версталась із використанням різних підходів: Flexbox CSS, Grid CSS і без використання Flexbox та Grid. Усі сторінки зберегли вихідний контент та функціонал.

1. Flexbox CSS: Сторінка, зверстана за допомогою Flexbox, дозволяє легко вирівнювати елементи як по горизонталі, так і по вертикалі. 
   Цей метод забезпечує гнучкість у розміщенні елементів навіть при зміні розмірів вікна браузера. Використання Flexbox є ефективним для створення адаптивного дизайну.

2. Grid CSS: У цьому екземплярі реалізована сіткова верстка за допомогою Grid Layout. Даний підхід надав можливість чітко розподілити елементи за заданими областями, що спростило створення складної структури сторінки. 
   Grid CSS зручний для роботи зі складними макетами, які вимагають точного позиціювання блоків.

3. Без Flexbox і Grid CSS: Цей варіант базується на використанні класичних способів верстки, таких як position, float і display: inline-block. 
   Він є менш гнучким і потребує більше зусиль для адаптації під різні розміри екранів. Однак цей метод корисний для підтримки старих браузерів або у випадках, коли сучасні підходи недоступні.

Для кожного екземпляра використовувались окремі CSS-файли: index.css, about.css, catalog.css, contact.css, faq.css, offers.css. Це забезпечило розділення стилів і полегшило управління кодом. 
Усі сторінки були перевірені на коректність роботи при різних розмірах вікна браузера та змінах контенту.
